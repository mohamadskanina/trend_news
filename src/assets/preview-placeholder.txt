PREVIEW IMAGE PLACEHOLDER

Replace this file with a preview.png image (1200x630px recommended) for Open Graph and Twitter Card previews.

The image should be:
- Format: PNG or JPG
- Dimensions: 1200x630 pixels (recommended)
- File size: Under 1MB for optimal loading
- Content: Should represent the Trend News App website

You can create this image using design tools like:
- Figma
- Adobe Photoshop
- Canva
- Online image editors

Once created, name it "preview.png" and place it in this directory (src/assets/preview.png).

Update the meta tags in HTML files to point to the correct URL:
<meta property="og:image" content="https://yourdomain.com/assets/preview.png">
